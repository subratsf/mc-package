###Usage

Example 1
```
%%=FormatNumber(1234.567,"C2","en-US")=%%
```

System returns this value:
```
$1234.57
```

Example 2
```
%%=FormatNumber(-12300099.45678,"N","fr-FR")=%%
```

System returns this value:
```
-12 300 099,46
```
